# EISEL — Site Institucional

Site do **Encontro de Inovação e Sustentabilidade na Engenharia Elétrica**, desenvolvido em HTML + Tailwind CSS via CDN. 
## Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Home — Hero, Sobre, Palestrantes, Patrocinadores, Local |
| `programacao.html` | Grade de atividades em timeline (Dia 1 e Dia 2) |
| `palestrantes.html` | Cards completos dos palestrantes e keynotes |
| `contato.html` | Formulário de contato + FAQ |
